# Atlas Portfolio Manager

### Professional Investment Tracking Dashboard

This project is a high-performance, enterprise-grade portfolio manager built for the Moroccan Stock Market (CSEMA).

## Features

- **Real-Time Dashboard**: Monitor your capital, P/L, and cash balance instantly.
- **Enterprise Visualization**: Interactive charts with 1M/3M/6M/1Y/ALL ranges, sparklines, and treemaps.
- **AI Analysis**: Integrated Google Gemini AI for portfolio insights and risk assessment.
- **Privacy First**: All data is stored locally in your browser (IndexedDB).

## Deployment

This project is configured for **Vercel**.

1. Import this repository to Vercel.
2. Add your **`GEMINI_API_KEY`** in the Environment Variables.
3. Deploy!

## Tech Stack

- **Vite** + **React** + **TypeScript**
- **TailwindCSS** for styling
- **Recharts** for visualizations
- **Dexie.js** for local database
